Chrome 构建错误

lld-link: error: invalid timestamp: -2142000. Expected 32-bit integer

build\util\LASTCHANGE.committime 文件内容为 0 导致的。 此文件里记录的应该是从 git 日志中提取的上次 commit 的时间戳，如果获取失败了默认填入 0。 手工改成合理的时间戳( epoch ) 再重新执行 gn 即可。
